Usage: grep [OPTION]... PATTERNS [FILE]...
Search for PATTERNS in each FILE. If filename isn't by user, program will search in each file in directory.

1) Highlight by word: grep -w '[WORD]' [FILENAME]
[EXAMPLE]:$ grep -w 'seven' test.txt

2) Highlight BY THE BEGININGD of the word:$ grep '\<[BEGINING]' [FILENAME]
[EXAPMLE]:$ grep '\<pi' ip.txt

3) Highlight BY THE END of the word:$ grep '[END]\>' [FILENAME]
[EXAPMLE]:$ grep 'ing\>' ip.txt

4) Highlight BY WORD IN THE BEGINNING of string:$ grep '^[WORD]' [FILENAME] 
[EXAMPLE]:$ grep "^64" ip.txt

5) Highlight BY WORD IN THE END of string:$ grep '[WORD]$' [FILENAME]
[EXAMPLE]:$ grep ".$" ip.txt

6) Highlight STRINGS ABOVE & BELLOW of needed:$ grep -C [COUNT(int)] '[WORD]' [FILENAME]
[EXAMPLE]:$ grep -C 1 '^1' ip.txt

7) Highlight ONLY STRING BELLOW needed: $ grep -A [COUNT(int)] '[WORD]' [FILENAME]
[EXAMPLE]:$ grep -A 1 '^1' ip.txt

7) Highlight ONLY STRING ABOVE needed: $ grep -B [COUNT(int)] '[WORD]' [FILENAME]
[EXAMPLE]:$ grep -A 1 '^1' ip.txt

8) Highlight STRINGS WITH NUMS IN RANGE:$ grep '[RANGE]' [FILENAME]
[EXAMPLE]:$ grep '[1-4]' ip.txt

9) Highlight STRINGS WITH SYMBOLS NOT IN RANGE:$ grep '[^RANGE]' [FILENAME]
[EXAMPLE]:$ grep '[^1-4]' ip.txt
 
10) ONLY IP's: grep -E "[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}" [FILENAME]

POSIX TABLE:
[:alpha:] Any alphabetical character, regardless of case
[:digit:] Any numerical character
[:alnum:] Any alphabetical or numerical character
[:blank:] Space or tab characters
[:xdigit:] Hexadecimal characters; any number or A–F or a–f
[:punct:] Any punctuation symbol
[:print:] Any printable character (not control characters)
[:space:] Any whitespace character
[:graph:] Exclude whitespace characters
[:upper:] Any uppercase letter
[:lower:] Any lowercase letter
[:cntrl:] Control characters

11) Highlight ONLY STRINGS WITH UPPER LETTERS: $ grep "[[:upper:]]" [FILENAME]

12) Operator OR in grep:$ grep -E "(word1|word2|...etc)" [FILENAME]
[EXAMPLE]:$ grep -E "(six|seven|eight)" ip.txt

13) Highlight strings, where WORD OCCURS SEVERAL TIMES: grep -E "(word1|word2|...).*\1" [FILENAME]
[EXAMPLE]:$ grep -E "(six|seven|eight).*\1" ip.txt
